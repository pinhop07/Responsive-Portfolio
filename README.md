# Responsive-Portfolio

### Overview

Responsive professional portfolio site using HTML/CSS. The websites design includes an aesthetically pleasing layout with different kinds of complex HTML elements. Three different media queries are used to make the website react to the size of the browser.

To try it out, open [Responsive Portfolio](https://pinhop07.github.io/Responsive-Portfolio/index.html) in your browser of choice.

### Technologies used

* HTML5
* CSS3

### Built With
